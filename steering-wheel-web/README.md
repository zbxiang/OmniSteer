# car-keyboard-management-system

方向盘产品管理系统

## 项目技术架构
- vue3
- vite
- element-plus
- sass
- typescript
- pinia
- vue-router
- eslint
- axios
- taiwindcss

## 项目依赖安装
```sh
pnpm install

## 项目运行
``` sh
pnpm dev

## 安装单个依赖：
```sh
pnpm add 包名

## 安装开发依赖：
```sh
pnpm add 包名 -D

## 删除依赖：
```sh
pnpm remove 包名

## 升级依赖：
```sh
pnpm up 包名

### 项目结构
- src
    - api/ # 接口层数据
    - assets/ # 静态资源
    - components/ # 通用组件
    - constants/ # 常量
    - enums/ # 枚举
    - router/ # 路由
    - stores/ # 状态管理
    - styles/ # 样式
    - types/ # 类型定义
    - utils/ # 工具
    - views/ # 页面
